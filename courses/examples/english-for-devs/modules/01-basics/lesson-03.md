# Control Flow

## Conditional Statements

- **If statement**: Runs code if a condition is `true`.
- **Else**: Runs if the condition is `false`.
- **Else if**: Checks a new condition.
- **Switch**: Checks multiple possible values.

## Loops (Bucles)

- **Loop**: Repeats a block of code.
- **Iterate**: To go through items one by one.
- **For loop**: Runs a specific number of times.
- **While loop**: Runs while a condition is `true`.
- **Break**: Stops the loop completely.
- **Continue**: Skips to the next iteration.
