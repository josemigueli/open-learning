# Error Handling

## Dealing with Errors

- **Try**: Block of code to test for errors.
- **Catch**: Block of code to handle the error.
- **Throw**: Create a custom error.
- **Exception**: An unexpected error event.

## Debugging (Depuración)

- **Bug**: An error in the code.
- **Debug**: The process of fixing bugs.
- **Breakpoint**: A paused spot in execution.
- **Stack trace**: The path of functions leading to the error.
- **Log**: Print information to the console.
