# Async Programming

## Key Concepts

- **Synchronous**: Code runs line by line, blocking execution.
- **Asynchronous (Async)**: Code runs in the background, non-blocking.
- **Callback**: A function passed as an argument to run later.

## Promises

- **Promise**: An object representing eventual completion.
- **Pending**: The initial state of a promise.
- **Resolved / Fulfilled**: The promise succeeded.
- **Rejected**: The promise failed.
- **Await**: Pause execution until a promise settles.
