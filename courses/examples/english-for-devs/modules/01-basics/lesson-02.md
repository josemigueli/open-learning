# Functions & Methods

## Key Vocabulary

- **Function**: A reusable block of code.
- **Method**: A function inside an object.
- **Parameter**: Variable in function definition.
- **Argument**: Actual value passed to function.
- **Return**: Send a value back.
- **Call / Invoke**: Execute the function.

## Common Action Verbs

- **Get**: Retrieve data (`getUser`).
- **Set**: Save data (`setAge`).
- **Fetch**: Get data from network (`fetchData`).
- **Parse**: Convert format (`JSON.parse`).
- **Handle**: Respond to event (`handleClick`).

## Reading Practice

> "A **function** executes when something **invokes** (calls) it. It takes **arguments** and can **return** a result."
