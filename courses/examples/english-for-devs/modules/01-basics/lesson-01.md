# Variables & Data Types

## Key Vocabulary

Learn these English terms for data types:

- **Variable**: A container for data.
- **String**: Text (e.g., `"Hello"`).
- **Integer**: Whole number (e.g., `42`).
- **Boolean**: `true` or `false`.
- **Array**: A list of items.

## Reading Practice

> "In programming, a **variable** is a container that stores data values. A **string** is used for storing text, while an **integer** stores whole numbers."

## Pro Tip

We use these English words even when speaking Spanish!

- Say "un string", not "una cadena".
- Say "un array", not "un arreglo".
