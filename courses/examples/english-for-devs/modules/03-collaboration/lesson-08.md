# Git Commands

## Basic Operations

- **Repository (Repo)**: The project folder managed by Git.
- **Commit**: Save changes to the local history.
- **Push**: Send local commits to a remote server.
- **Pull**: Fetch and download changes from remote.

## Branching

- **Branch**: A separate line of development.
- **Merge**: Combine changes from two branches.
- **Conflict**: When Git cannot automatically merge changes.
- **Resolve**: To fix a merge conflict manually.
