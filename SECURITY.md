# Security Policy

## Supported Versions

Only the latest version of this project is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please **DO NOT** create a public Issue. Instead, follow these steps:

1. **Use GitHub's Private Reporting:** Go to the **Security** tab of this repository and click on **Report a vulnerability**. This will open a private discussion with the maintainers.
2. **Describe:** Include as much detail as possible (steps to reproduce, impact, and suggested fix if you have one).
3. **Wait:** Give us some time to respond before taking any further action.

We appreciate your help in keeping this project secure!
